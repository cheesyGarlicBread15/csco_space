# CSCo Space

The official website of **CSCo** — the **College of Information Sciences and Computing (CISC) Student Council Organization** of Central Mindanao University.

The site introduces the organization, publishes its mission and vision, and puts a face to every officer, adviser, and volunteer serving the student body, so students know who represents them and how to reach them.

## Getting Started

Requires Node.js 20+ and npm.

```bash
git clone https://github.com/cheesyGarlicBread15/csco_space.git
```

```bash
npm install
```

```bash
npm run dev
```

## Project Structure

```
src/
├── assets/         # logo and profile images
├── components/     # header, footer, profile card, UI primitives
├── pages/          # Home, Team, and the shared layout
├── utils/          # image loading helpers
└── main.tsx        # router setup
```
